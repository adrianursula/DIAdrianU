# 🎯 Trello Clone - ProjectFlow

Un clon completo de Trello construido con React, TypeScript y Zustand. Incluye todas las funcionalidades principales de gestión de proyectos y colaboración en equipo.

## 🚀 Demo en Vivo

- **Aplicación:** [https://tuusuario.github.io/versiones/app/](https://tuusuario.github.io/versiones/app/)
- **Documentación:** [https://tuusuario.github.io/versiones/docs/](https://tuusuario.github.io/versiones/docs/)

## ✨ Características Principales

### Gestión de Tableros
- 📋 Tableros múltiples con diferentes fondos
- 🎨 Fondos personalizables (colores e imágenes)
- ⭐ Sistema de favoritos
- 🔍 Búsqueda inteligente de tareas y tableros

### Organización de Tareas
- 📝 Creación y edición de tarjetas
- 🏷️ Sistema de etiquetas personalizables
- 📅 Fechas de vencimiento con indicadores visuales
- ✅ Checklists y subtareas
- 📎 Adjuntos y portadas de tarjetas
- 💬 Comentarios y actividad

### Colaboración
- 👥 Espacios de trabajo
- 👤 Sistema de usuarios y permisos
- 🔐 Autenticación con login/registro
- 🛡️ Roles de administrador
- 📧 Invitación de miembros

### Automatización
- ⚡ Reglas de automatización Butler-style
- 🔄 Acciones automáticas al mover tarjetas
- 🎯 Triggers personalizables

### Vistas
- 📊 Vista Kanban (tablero)
- 📅 Vista de calendario
- 🔀 Drag & drop completo
- 📱 Diseño responsive

### Funcionalidades Avanzadas
- 🌙 Modo oscuro
- 🎨 Colores de acento personalizables
- ⌨️ Atajos de teclado
- 📋 Plantillas de tableros
- 📈 Estadísticas y analíticas

## 🛠️ Tecnologías Utilizadas

- **Framework:** React 19.2.0
- **Lenguaje:** TypeScript 5.9.3
- **Estado:** Zustand 5.0.9 con persistencia
- **Drag & Drop:** @dnd-kit/core
- **Estilos:** TailwindCSS 3.4.18
- **Build:** Vite 7.2.4
- **Iconos:** Lucide React

## 📦 Instalación y Desarrollo Local

```bash
# Clonar el repositorio
git clone https://github.com/tuusuario/versiones.git

# Navegar a la carpeta del proyecto
cd versiones/trello-final

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```

## 🔧 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Compila para producción
- `npm run preview` - Previsualiza el build de producción
- `npm run lint` - Ejecuta el linter

## 🎓 Desarrollo Iterativo

Este proyecto fue desarrollado iterativamente utilizando IA (Gemini), pasando por más de 40 versiones desde un gestor de tareas básico hasta un clon completo de Trello.

### Fases principales:
1. **Serie 1.x** - Refactorización y setup inicial
2. **Serie 2.x** - Interactividad y diseño
3. **Serie 3.x** - Multi-tablero y arquitectura
4. **Serie 4.x** - Ecosistema completo (espacios, plantillas)
5. **Serie 5.x** - Sistema de usuarios y autenticación
6. **Serie 6.x** - Optimización con Zustand
7. **Serie 7.x** - Pulido final y GitHub deployment
