Registro de Cambios (Changelog)

Documentación detallada de la evolución de Trello Clone (ProjectFlow), desde una aplicación monolítica básica hasta un ecosistema multi-usuario y multi-espacio completo.

Este documento ha sido generado dinámicamente basado en la conversación de desarrollo.

Versión 4.0: Arquitectura "Full Clone"

Prompt

"basado en este prompt y a partir de esta version de la aplicacion, copia la aplicacion de Trello como un clon de la aplicacion, ve poco a poco..."

ARQUITECTURA Refactorización Masiva

Se migró de un estado local simple a un AppContext global complejo. Se implementó un enrutamiento simulado para manejar vistas sin recargas y se diseñó el Sidebar colapsable al estilo Trello.

Versión 4.2: Layout y Funcionalidad Core

Prompt

"vale, esta sera la version 4.2, elimina el scroll horizontal y vamos implementando funcionalidad a todo, ya que ahora mismo es hay muchos botones sin funcionalidad"

FEATURE Funcionalidad de Botones

Conexión de la lógica para los botones "Mover", "Copiar" y "Archivar" en el modal de tareas. Implementación real del menú de acciones de lista (Ordenar, Eliminar).

FIX Layout Robusto

Corrección del desbordamiento horizontal (scroll) y ajuste del layout 100dvh para móviles.

Versión 4.4: Detalles de Interacción

Prompt

"implementa otra funcionalidad" (Enfoque en edición y etiquetas)

FEATURE Edición en Línea

Capacidad para renombrar listas haciendo clic directamente en el título.

FEATURE Selector de Etiquetas

Implementación de un Popover real para añadir/quitar etiquetas de colores a las tareas.

Versión 4.5: Modal Rico

Prompt

"implementa otra funcionalidad" (Enfoque en adjuntos y fechas)

FEATURE Sistema de Adjuntos

Capacidad para subir archivos reales (simulado en memoria), visualizarlos en lista y usar imágenes como Portada (Cover) de la tarjeta.

FEATURE Fechas de Vencimiento

Integración de selector de fecha nativo y lógica visual para fechas cumplidas (verde) o pendientes.

Versión 4.6 / 4.7: Sistema de Plantillas

Prompt

"implementa ahora las plantillas por favor" y posteriormente "ahora que pueda acceder mediante su boton..."

FEATURE Galería de Plantillas

Nueva vista dedicada con tableros pre-configurados (Gestión de Proyectos, Kanban Simple, etc.).

LOGIC Instanciación

Lógica para clonar una plantilla completa (columnas, tareas, etiquetas) en un nuevo tablero funcional.

Versión 4.8 / 4.9: Espacios de Trabajo

Prompt

"implementa los espacios de trabajo" y "ahora como puedo acceder a ese espacio de trabajo?"

ARQUITECTURA Jerarquía de Datos

Introducción de la entidad Workspace. Los tableros ahora pertenecen a un espacio, no flotan libremente.

FEATURE Vista de Espacio

Nueva página dashboard que muestra los tableros agrupados en cuadrícula, miembros del espacio y configuración.

Versión 4.10: Buscador Contextual

Prompt

"para que se usa el buscador?, por que quiero buscar algo y no funciona"

FIX Lógica de Búsqueda

Se arregló el buscador para que sea sensible al contexto: filtra tableros si estás en el Espacio de Trabajo, y filtra tareas si estás dentro de un Tablero.

Versión 5.0 / 5.2: Usuarios y Aislamiento

Prompt

"añade el sistema de perfiles y usuarios" y "haz que con cada perfil, sea diferente para cada usuario"

FEATURE Cambio de Usuario

Capacidad para cambiar entre usuarios predefinidos (Admin, Ana, Carlos, Luisa) instantáneamente.

SEGURIDAD Segregación de Datos

Implementación de filtros estrictos. Los usuarios ahora solo ven los espacios y tableros a los que han sido invitados.

Versión 5.3: Roles y Permisos

Prompt

"haz que el admin pueda ver todo de todos y luego cada usuario propio solo pueda ver sus tableros..."

FEATURE Superadmin

Creación del rol isAdmin. El usuario Admin tiene acceso de lectura/escritura global a todos los espacios, ignorando las restricciones de membresía.

Versión 5.3.2 / 5.3.3: Autenticación Real

Prompt

"añade una ventana de inicio de sesion... y luego para registrar un nuevo usuario..."

FEATURE Flujo Auth

Pantalla de Login y Registro funcional. Validación de credenciales y creación de nuevos usuarios en tiempo real.

FIX Estabilidad

Corrección de errores críticos en la inicialización del estado (useAppWrapper error) y lógica de creación automática de espacios personales al registrarse.

Versión 5.5: App Switcher

Prompt

"...solo implementa lo del menu de los 9 puntos..."

FEATURE Menú Atlassian

Implementación del menú desplegable de 9 puntos en el header, simulando la navegación entre productos de Atlassian (Jira, Confluence, Bitbucket).

Versión 5.5.1: Hotfix

Prompt

"arregla este error: The symbol 'AppSwitcher' has already been declared"

FIX Compilación

Eliminación de código duplicado generado accidentalmente en la versión anterior para restaurar la estabilidad de la aplicación.
